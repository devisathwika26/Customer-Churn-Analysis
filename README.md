Project Title:

Customer Churn Analysis using Artificial Intelligence

Project Scope:

This project uses Artificial Intelligence and Machine Learning techniques to predict whether a customer will leave (churn) a telecom company based on customer behavior and service usage data.

Technologies Used:

• Python
• Google Colab
• Scikit-learn
• Pandas
• NumPy

Dataset:

Telco Customer Churn Dataset (Real-world dataset)

It contains features such as:

•Gender
•SeniorCitizen
•Tenure
•InternetService
•Contract Type
•MonthlyCharges
•TotalCharges
•Churn (Target variable)

Methodology:

1.Loaded the dataset.

2.Removed irrelevant columns like Customer ID.

3.Handled missing values in TotalCharges.

4.Encoded categorical variables into numerical format.

5.Split dataset into training and testing sets.

6.Trained a Random Forest Classifier model.

7.Predicted customer churn.

8.Evaluated model performance using accuracy and classification report.

Output:

• Accuracy: ~80%
• Classification Report: Precision, Recall, and F1-score for both classes
• Predicted Result: Customer Will Churn
